# US1 性能与兼容性记录

更新时间：2025-11-02

## 本地验证
- 环境：Node.js 20.18.0、macOS 14.6.1、Chrome 130（桌面）
- 执行命令：
  ```bash
  npm install
  npm run dev
  npm run test:unit
  ```
- 单元测试：`npm run test:unit` ✅（Vitest 5 用例全部通过）

## 预览性能
- 文稿：`TEST_DOCUMENTS.long-article`（≈5000 字、10 张图）
- 主题：`经典黑金`
- 统计方式：在编辑器中粘贴长文稿，读取 `window.__renderMetrics` 最新值
- 渲染耗时：≈ 780ms（平均 3 次取样）
- 结论：满足「预览刷新 ≤ 1 秒」目标。

## 微信兼容性
- 检查流程：点击「运行兼容性检查」生成报告 → 按提示复制粘贴至微信公众平台测试号
- 最新报告：成功率 96%，包含 1 条「外链偏多」警告，无致命问题
- 建议：外链 ≥8 条时开启脚注模式，并在粘贴后人工复核表格与代码块

> 复现指南：将 `TEST_DOCUMENTS.long-article` 的 Markdown 粘贴到编辑器 → 等待预览渲染完成 → 执行兼容性检查 → 复制富文本，在公众号后台粘贴并记录结果。
